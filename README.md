# 💫 About Me:
Python Backend-разработчик с опытом более 3 лет в проектировании и разработке масштабируемых веб-приложений и автоматизации бизнес-процессов. Специализируюсь на создании высоконагруженных REST API с использованием Django и FastAPI. Имею опыт в организации фоновой обработки задач (Celery, Taskiq), кэшировании данных (Redis) и оптимизации реляционных баз данных (PostgreSQL, MySQL). Уверенно разворачиваю и сопровождаю сервисы в production-среде с использованием Docker, Nginx и Linux.<br>
