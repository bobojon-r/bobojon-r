# 💫 About Me:
Python Backend-разработчик с опытом более 3 лет в проектировании и разработке
масштабируемых веб-приложений и автоматизированных систем. Специализируюсь на
создании микросервисных архитектур и высоконагруженных REST API.

Имею практический опыт построения event-driven систем на Apache Kafka,
контейнеризации и оркестрации сервисов в Kubernetes (minikube + AWS EKS),
настройки CI/CD пайплайнов через GitHub Actions и развёртывания облачной
инфраструктуры через Terraform (AWS: EKS, RDS, MSK, ECR). Интегрировал платёжные
системы (Stripe), настраивал observability стек (Prometheus, Grafana, OpenTelemetry,
Sentry) и разрабатывал Telegram-ботов на aiogram.
